# Collider Simulator
Collider simulator for TAU Physics Laboratory C Course

## List of available particels
* photon
* electron
* positron
* muon
* mu-plus
* neutrino
* pi-0
* pi-plus
* pi-minus
* k-long
* k-short
* k-plus
* k-minus
* neutron
* proton
* antiproton
* lambda
* antilambda
* sigma-plus
* sigma-minus
* sigma-0
